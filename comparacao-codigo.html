Comparação de Código - Lista de Tarefas

Código Antigo

const tarefa = { id: Date.now(), texto: texto, completa: false, dataCriacao: new Date() };

Código Novo

const tarefa = { id: Date.now(), texto: texto, completa: false, dataCriacao: new Date(), dataVencimento: dataVencimento, categoria: categoria, subtarefas: [] };

Principais Diferenças:

  • Adicionado campo dataVencimento para controlar o prazo da tarefa
  • Adicionado campo categoria para classificar as tarefas
  • Adicionado array subtarefas para permitir tarefas aninhadas

Explicação:

O objeto tarefa foi expandido para incluir novas funcionalidades:

  • dataVencimento: Permite definir um prazo para a tarefa
  • categoria: Permite classificar as tarefas em diferentes tipos
  • subtarefas: Array que armazena subtarefas relacionadas

Função Antiga - Adicionar Tarefa

function adicionarTarefa() { const texto = tarefaInput.value.trim(); if (texto === '') { alert('Por favor, digite uma tarefa!'); return; } const tarefa = { id: Date.now(), texto: texto, completa: false, dataCriacao: new Date() }; tarefas.push(tarefa); tarefaInput.value = ''; atualizarLista(); salvarTarefas(); }

Função Nova - Adicionar Tarefa

function adicionarTarefa() { const texto = tarefaInput.value.trim(); const dataVencimento = document.getElementById('data-vencimento').value; const categoria = document.getElementById('categoria').value; if (texto === '') { alert('Por favor, digite uma tarefa!'); return; } const tarefa = { id: Date.now(), texto: texto, completa: false, dataCriacao: new Date(), dataVencimento: dataVencimento, categoria: categoria, subtarefas: [] }; tarefas.push(tarefa); tarefaInput.value = ''; document.getElementById('data-vencimento').value = ''; document.getElementById('categoria').value = 'pessoal'; atualizarLista(); salvarTarefas(); }

Novas Funções Adicionadas

function adicionarSubtarefa(tarefaId, texto) { const tarefa = tarefas.find(item => item.id === tarefaId); if (tarefa) { const subtarefa = { id: Date.now(), texto: texto, completa: false }; tarefa.subtarefas.push(subtarefa); atualizarLista(); salvarTarefas(); } } function completarSubtarefa(tarefaId, subtarefaId) { const tarefa = tarefas.find(item => item.id === tarefaId); if (tarefa) { const subtarefa = tarefa.subtarefas.find(sub => sub.id === subtarefaId); if (subtarefa) { subtarefa.completa = !subtarefa.completa; atualizarLista(); salvarTarefas(); } } } function atualizarEstatisticas() { const total = tarefas.length; const completas = tarefas.filter(tarefa => tarefa.completa).length; const pendentes = total - completas; sumarioEl.textContent = `Total de tarefas: ${total} | Completas: ${completas} | Pendentes: ${pendentes}`; const porcentagem = total > 0 ? (completas / total) * 100 : 0; document.getElementById('barra-progresso').style.width = `${porcentagem}%`; const estatisticasTexto = document.getElementById('estatisticas-texto'); const hoje = new Date(); const tarefasHoje = tarefas.filter(tarefa => { const dataCriacao = new Date(tarefa.dataCriacao); return dataCriacao.toDateString() === hoje.toDateString(); }).length; estatisticasTexto.textContent = `Tarefas criadas hoje: ${tarefasHoje}`; }

Resumo das Novas Funcionalidades

  1. Data de Vencimento: Permite definir prazos para as tarefas
  2. Categorias: Organiza tarefas em grupos (Pessoal, Trabalho, Estudos)
  3. Subtarefas: Permite criar tarefas aninhadas
  4. Estatísticas: Mostra progresso e métricas de produtividade
  5. Ordenação: Permite organizar tarefas por diferentes critérios
